This demo utilizes a modified version of Apex Charts v3.22.1, where the PNG and SVG file names are set in the chart options object.

The modified source can be found here.

You can change the file name by enter text in the box below and then clicking the button. To see the file name use the export menu and choose one of the types. When the file is ready to download you should see the name you have chosen.


Uncheck/check boxes to disable or enable file download types.
